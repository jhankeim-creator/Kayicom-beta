## üìù DNS Records to Add

### For Backend API (api.kayicom.com):

#### 1. CNAME Record:
```
Type: CNAME
Name: api
Value: [COPY FROM RAILWAY BACKEND CNAME]
TTL: 1 Hour (or Auto)
```

#### 2. TXT Record (for SSL):
```
Type: TXT
Name: _acme-challenge.api
Value: [COPY FROM RAILWAY BACKEND TXT]
TTL: 1 Hour (or Auto)
```

---

### For Frontend (kayicom.com):

#### 1. CNAME Record:
```
Type: CNAME
Name: @

TTL: 1 Hour (or Auto)

#### 2. TXT Record (for SSL):
```
Type: TXT
